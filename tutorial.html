How to install

1. Click here to download

2. Unzip this file

3. Go to GoogleChrome click more tools and select Extensions to open Extension window

4. In the Extensions tab, enable Developer mode

5. Then, Click Load unpacked to open explore window

6. Find the OldInterface folder and choose this

7. Click this to pin extension

8. Go to facebook.com and click enable to use Facebook old layout
